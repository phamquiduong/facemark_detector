# Facemark detector

# Create venv
- Windows
```
python -m venv venv
```
- Ubuntu
```
sudo apt install python3-pip python3-venv
python3 -m venv venv
```

# Active this venv
- Windows
```
.\venv\Scripts\activate
```
- Linux/Mac
```
source venv\bin\active
```

# Install library
```
pip install -r requirements.txt
```

# Run this source
```
python detector.py
```
